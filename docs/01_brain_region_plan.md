## 一、总体解剖架构

人脑按胚胎发育来源分为 **前脑、中脑、后脑** 三大部分：

```
人脑 (Brain)
│
├── 前脑 (Forebrain / Prosencephalon)
│   ├── 端脑 (Telencephalon) ─── 占脑体积 ~85%
│   │   ├── 大脑皮层 (Neocortex) ─── 额叶/顶叶/颞叶/枕叶/岛叶/扣带回
│   │   ├── 基底核 (Basal Ganglia) ─── 纹状体/苍白球
│   │   ├── 海马体 (Hippocampus) ─── 记忆编码与巩固
│   │   ├── 杏仁核 (Amygdala) ─── 情感标记与优先级
│   │   ├── 隔核 (Septal Nuclei) ─── theta节律/奖赏
│   │   └── 屏状核 (Claustrum) ─── 跨模态整合(?)
│   │
│   └── 间脑 (Diencephalon) ─── 被端脑包裹
│       ├── 丘脑 (Thalamus) ─── 中枢路由器
│       ├── 下丘脑 (Hypothalamus) ─── 内驱力/睡眠/体温/激素
│       ├── 上丘脑 (Epithalamus) ─── 缰核(负奖励) + 松果体(昼夜节律)
│       └── 底丘脑 (Subthalamus) ─── STN(全局刹车)
│
├── 中脑 (Midbrain / Mesencephalon) ─── 连接前脑与后脑
│   ├── 顶盖 (Tectum) ─── 上丘(视觉定向) + 下丘(听觉中继)
│   └── 被盖 (Tegmentum) ─── VTA(DA) + 黑质 + 红核 + PAG + 中缝核(5-HT)
│
└── 后脑 (Hindbrain / Rhombencephalon)
    ├── 脑桥 (Pons) ─── 蓝斑核(NE) + 皮层-小脑中继
    ├── 小脑 (Cerebellum) ─── 预测与误差校正
    └── 延髓 (Medulla) ─── 呼吸/心跳(基础生存，低优先级)
```

## 二、脑区详细分区表

> 按真实解剖层级组织: **前脑(端脑+间脑) → 中脑 → 后脑**

---

### 2.1 前脑 — 端脑 (Telencephalon)

端脑是人脑最大的部分 (~85% 体积)，包含大脑皮层和皮层下核团。

#### 2.1.1 大脑皮层 (Neocortex) — 6层结构处理核心

新皮层占大脑表面积的 ~76%，是高级认知的物质基础。所有皮层区域共享统一的 **6 层柱状结构**。

> **兴奋性/抑制性神经元比例**: 全皮层平均约 80-85% 兴奋性、15-20% 抑制性，但不同脑区和层次有较大差异：
> PFC 抑制性比例更高 (~25-30%，支持更精确的竞争选择)；V1 接近平均值 (~20%)；
> 海马 CA1 抑制性比例约 10-15%。L4 层的抑制性比例通常低于 L2/3 和 L5。
> (参考: PNAS 2021, Frontiers Neural Circuits 2021)

> **注意**: 同一 Brodmann 区可能被多个功能脑区共享（如 BA39 同时涉及 PPC、TPJ 和角回），这是正常的解剖学现象。在 悟韵 (WuYun) 实现中，应使用唯一编号（如 S-04、A-06）而非 Brodmann 区编号来标识计算模块。下表仅列出主要功能脑区的代表性区域，未覆盖全部 ~160 亿皮层神经元。

**基底神经节三条通路：**

```
直接通路 (Go):    皮层 → 纹状体(D1) → GPi/SNr → 丘脑 → 皮层
                  效果: 去抑制 → 促进选中的动作

间接通路 (NoGo):  皮层 → 纹状体(D2) → GPe → STN → GPi/SNr → 丘脑
                  效果: 增强抑制 → 抑制竞争动作

超直接通路 (Stop): 皮层 → STN → GPi/SNr → 丘脑
                   效果: 全局暂停 → 等待更多信息

奖赏通路:          NAcc → 腹侧苍白球(VP) → 丘脑MD → PFC
                   效果: 奖赏信号影响认知
```

**海马体关键处理流程：**

```
感觉信息 → EC(L2) → DG(模式分离) → CA3(模式补全/存储)
                                         ↓
皮层 ← EC(L5) ← Subiculum ← CA1(比较/输出) ←┘
                                    ↑
                              EC(L3) → CA1 (直接通路)

记忆巩固(睡眠期): CA3/CA1 → 通过Sharp Wave Ripples → 新皮层(长期存储)
```

**核心参数：**
- 齿状回稀疏度: ~2% 激活率（极高的模式分离能力）
- CA3 循环连接: 约 1-2% 的 CA3→CA3 连接概率（自联想网络，每个 CA3 细胞接收约 1.2×10⁴ 个循环侧枝突触，Rolls 2013; Le Duigou et al. 2014）
- 记忆容量: 理论估算约 ~3.6×10⁴ 个独立模式 (Treves & Rolls 1991，基于 CA3 自联想网络理论，实际容量受稀疏度和编码方式影响)
- 巩固周期: 新记忆 → 海马依赖期数周到数月（某些情景记忆可达数年）→ 皮层独立 (Squire & Alvarez 1995)

#### 2.1.5 隔核 (Septal Nuclei) — theta节律与奖赏

> **解剖位置**: 端脑内侧，胼胝体下方，海马前方。
> **v0.2 新增**

|------|---------|---------|---------|--------------|

**神经调质关键参数映射：**

```
多巴胺 (DA):
  正信号 (VTA/SNc):
    - 相位性释放 → reward_prediction_error (RPE > 0)
    - 紧张性水平 → motivation_level / exploration_bonus
    - D1受体 → 增强 Go 通路 (促进)
    - D2受体(Gi/o耦合) → 降低 D2-MSN 兴奋性 → 减弱 NoGo 信号 → 净效果为促进动作
    - DA 还通过突触前 D2 受体降低皮层→纹状体谷氨酸释放 (Surmeier et al. 2007)
  负信号 (LHb → VTA抑制):
    - 预期奖励未出现 → LHb兴奋 → 抑制VTA DA释放 → RPE < 0
    - 驱动回避学习和行为修正

去甲肾上腺素 (NE):
  - 低水平 → 困倦/不专注
  - 中等水平 → 聚焦注意力 (exploitation)
  - 高水平 → 高警觉/广泛注意力 (exploration)
  - 映射 → temperature 参数 (探索-利用权衡)

5-羟色胺 (5-HT):
  - 高水平 → 耐心/等待远期奖励
  - 低水平 → 冲动/偏好即时奖励
  - 映射 → discount_factor γ

乙酰胆碱 (ACh):
  - 高水平 → 增强自下而上处理 (learning_mode)
  - 低水平 → 增强自上而下处理 (inference_mode)
  - 映射 → learning_rate + signal_noise_ratio
```

### 3.1 主要通路汇总

```
信号流向图 (→ 为前馈, ⟷ 为双向, ⟶ 为调制, --- 为抑制)

[感觉输入]
    │
    ├──→ [上丘] → 快速定向(皮层下捷径)
    │
    ▼
[丘脑] ⟷ [初级感觉皮层] → [次级感觉皮层] → [联合皮层]
    ▲           │                                  │
    │           └──────────────────┐                │
    │                              ▼                ▼
[TRN]◄──────────────────── [后顶叶/颞叶] ⟷ [前额叶(PFC)]
                                  │                │
                                  ▼                ▼
                       [隔核]→θ→[海马体] ⟷ [EC]   [基底神经节]
                                  │                │
                                  ▼                ▼
                        [乳头体→丘脑AN→扣带回]  [丘脑→皮层(去抑制)]
                        (Papez记忆回路)             │
                                                   ▼
                                             [运动皮层] → [运动输出]
                                                   ▲
                                                   │
                                              [小脑(校正)]

奖赏/惩罚系统:
  VTA(+RPE) ⟶ PFC, NAcc, 纹状体 (多巴胺: 正奖励信号)
  LHb(-RPE) ---| VTA (抑制DA释放: 负奖励信号)

调制系统:
  LC  ⟶ 全脑 (NE: 警觉/探索-利用)
  Raphe ⟶ 全脑 (5-HT: 耐心/时间折扣)
  BF  ⟶ 皮层, 海马体 (ACh: 学习/注意力)

内驱力系统:
  [下丘脑] → 饥饿/渴/体温/睡眠信号 → 影响BG动作选择优先级
  [下丘脑VLPO] ⟷ [外侧下丘脑orexin] (睡眠-觉醒翻转开关)
  [下丘脑PVN] → HPA轴 → 应激响应 → 影响NE/DA释放
```

### 4.2 皮层柱内部 6 层结构规格

```
每个皮层柱 (CorticalColumn) 的统一内部结构:

┌─────────────────────────────────────────────┐
│ L1 - 分子层 (Molecular Layer)                │
│   功能: 接收远程反馈调制                       │
│   输入: 高层皮层的 top-down 预测               │
│   计算: 调制性加权 (不生成脉冲)                │
│   神经元类型: 极少(主要是轴突和树突)            │
├─────────────────────────────────────────────┤
│ L2/3 - 浅层锥体层 (Superficial Pyramidal)     │
│   功能: 特征关联 + 跨柱通信                    │
│   输入: L4 前馈 + L1 调制 + 侧向连接           │
│   计算: 预测误差计算 + 侧向竞争抑制             │
│   输出: → 同层其他柱(侧向), → 高层柱(前馈)     │
│   神经元: 兴奋性锥体 + 抑制性中间神经元          │
├─────────────────────────────────────────────┤
│ L4 - 颗粒层 (Granular Layer)                  │
│   功能: 主要输入层                             │
│   输入: 丘脑前馈信号 / 低层皮层前馈              │
│   计算: 局部特征提取                           │
│   输出: → L2/3 (向上), → L6 (向下)            │
│   神经元: 星形细胞(兴奋性) + 抑制性中间神经元    │
├─────────────────────────────────────────────┤
│ L5 - 深层锥体层 (Deep Pyramidal)              │
│   功能: 主要输出层 (驱动行为/皮层下)             │
│   输入: L2/3 + L6 + 直接丘脑输入               │
│   计算: 输出整合 + burst模式(注意力标记)        │
│   输出: → 基底神经节, → 脑干, → 脊髓            │
│         → 高层皮层(驱动性前馈)                  │
│   神经元: 大型锥体细胞 + 抑制性中间神经元        │
├─────────────────────────────────────────────┤
│ L6 - 多形层 (Polymorphic Layer)               │
│   功能: 反馈输出层 (→ 丘脑调制)                │
│   输入: L4 + L5 + 高层反馈                     │
│   计算: 生成 top-down 预测信号                  │
│   输出: → 丘脑(反馈调制), → 低层皮层L1          │
│   神经元: 多形锥体细胞 + 抑制性                 │
└─────────────────────────────────────────────┘
```
