## 一、总体架构分层

人脑可按 **功能层级** 划分为以下四大系统：

```
┌─────────────────────────────────────────────────────┐
│             Level 4: 执行控制系统                      │
│         前额叶皮层 (PFC) · 前扣带回 (ACC)              │
├─────────────────────────────────────────────────────┤
│             Level 3: 高级认知系统                      │
│    联合皮层 (Association Cortex) · 布洛卡/韦尼克区      │
├─────────────────────────────────────────────────────┤
│             Level 2: 中枢调控系统                      │
│  丘脑 · 基底神经节 · 海马体 · 杏仁核 · 小脑             │
├─────────────────────────────────────────────────────┤
│             Level 1: 感知-运动系统                     │
│    初级视觉皮层 · 初级听觉皮层 · 初级运动皮层            │
└─────────────────────────────────────────────────────┘
```

## 二、脑区详细分区表

### 2.1 新皮层 (Neocortex) —— 6 层结构处理核心

新皮层占大脑表面积的 ~76%，是高级认知的物质基础。所有皮层区域共享统一的 **6 层柱状结构**。

> **兴奋性/抑制性神经元比例**: 全皮层平均约 80-85% 兴奋性、15-20% 抑制性，但不同脑区和层次有较大差异：
> PFC 抑制性比例更高 (~25-30%，支持更精确的竞争选择)；V1 接近平均值 (~20%)；
> 海马 CA1 抑制性比例约 10-15%。L4 层的抑制性比例通常低于 L2/3 和 L5。
> (参考: PNAS 2021, Frontiers Neural Circuits 2021)

> **注意**: 同一 Brodmann 区可能被多个功能脑区共享（如 BA39 同时涉及 PPC、TPJ 和角回），这是正常的解剖学现象。在 悟韵 (WuYun) 实现中，应使用唯一编号（如 S-04、A-06）而非 Brodmann 区编号来标识计算模块。下表仅列出主要功能脑区的代表性区域，未覆盖全部 ~160 亿皮层神经元。

### 2.2 皮层下结构 (Subcortical Structures)

#### 2.2.1 丘脑 (Thalamus) —— 中枢路由器

| 编号 | 核团名称 | 生物功能 | 计算映射 | 神经元数量(估) | 连接目标 |
|------|---------|---------|---------|--------------|---------|
| T-01 | 外侧膝状体 (LGN) | 视觉中继 | 视觉信号门控+路由 | ~0.1亿 | V1, V2 |
| T-02 | 内侧膝状体 (MGN) | 听觉中继 | 听觉信号门控+路由 | ~0.05亿 | A1 |
| T-03 | 腹后外侧核 (VPL) | 躯体感觉中继 | 触觉信号路由 | ~0.05亿 | S-07 |
| T-04 | 丘脑枕 (Pulvinar) | 视觉注意力调制 | 注意力聚光灯 | ~0.08亿 | V1-V5, PPC |
| T-05 | 背内侧核 (MD) | 前额叶中继 | 认知信号路由 | ~0.06亿 | PFC, OFC |
| T-06 | 丘脑网状核 (TRN) | 抑制性门控 | 全局抑制/选择门控 | ~0.04亿 | 所有丘脑核团 |
| T-07 | 前核群 (AN) | 记忆/导航中继 | 海马-皮层中继 | ~0.03亿 | 海马体, ACC |
| T-08 | 腹后内侧核 (VPM) | 面部感觉/味觉中继 | 面部触觉+味觉信号路由 | ~0.04亿 | S-07, S-09 |

**丘脑关键特性：**
- 所有感觉信号（嗅觉除外）必须经过丘脑中继才能到达皮层
- TRN (网状核) 是唯一的纯抑制性核团，控制所有其他核团的门控
- 双向连接：皮层→丘脑的反馈投射约是丘脑→皮层前馈投射的 **10倍**

#### 2.2.2 基底神经节 (Basal Ganglia) —— 决策与强化学习引擎

| 编号 | 核团名称 | 生物功能 | 计算映射 | 神经元数量(估) | 信号类型 |
|------|---------|---------|---------|--------------|---------|
| BG-01 | 纹状体-壳核 (Putamen) | 运动选择 | 动作候选评估 | ~0.3亿 | 抑制性(GABA) |
| BG-02 | 纹状体-尾状核 (Caudate) | 认知选择 | 认知策略评估 | ~0.2亿 | 抑制性(GABA) |
| BG-03 | 伏隔核 (NAcc) | 奖赏/动机 | 奖励信号聚合 | ~0.05亿 | 抑制性(GABA) |
| BG-04 | 苍白球外侧 (GPe) | 间接通路中继 | NoGo 信号放大 | ~0.05亿 | 抑制性(GABA) |
| BG-05 | 苍白球内侧 (GPi) | 输出核-运动 | 最终动作门控 | ~0.03亿 | 抑制性(GABA) |
| BG-06 | 黑质网状部 (SNr) | 输出核-眼动/认知 | 认知行为门控 | ~0.03亿 | 抑制性(GABA) |
| BG-07 | 丘脑底核 (STN) | 超直接通路-全局刹车 | 紧急停止/冲突暂停 | ~0.02亿 | 兴奋性(Glu) |

**基底神经节三条通路：**

```
直接通路 (Go):    皮层 → 纹状体(D1) → GPi/SNr → 丘脑 → 皮层
                  效果: 去抑制 → 促进选中的动作

间接通路 (NoGo):  皮层 → 纹状体(D2) → GPe → STN → GPi/SNr → 丘脑
                  效果: 增强抑制 → 抑制竞争动作

超直接通路 (Stop): 皮层 → STN → GPi/SNr → 丘脑
                   效果: 全局暂停 → 等待更多信息
```

#### 2.2.3 海马体 (Hippocampus) —— 记忆编码与巩固

| 编号 | 亚区名称 | 生物功能 | 计算映射 | 神经元数量(估) | 特殊机制 |
|------|---------|---------|---------|--------------|---------|
| H-01 | 内嗅皮层 (EC) | 海马输入/输出接口 | 记忆地址编码 | ~0.1亿 | 网格细胞 |
| H-02 | 齿状回 (DG) | 模式分离 | 输入正交化/去相关 | ~0.1亿 | 极稀疏激活(~2%) |
| H-03 | CA3 | 模式补全/自联想 | 联想记忆回忆 | ~0.03亿 | 强循环连接 |
| H-04 | CA1 | 模式比较/输出 | 新旧信息比较 | ~0.04亿 | 失匹配检测 |
| H-05 | 海马下托 (Subiculum) | 海马主要输出 | 记忆信号分发 | ~0.03亿 | 多目标投射 |

**海马体关键处理流程：**

```
感觉信息 → EC(L2) → DG(模式分离) → CA3(模式补全/存储)
                                         ↓
皮层 ← EC(L5) ← Subiculum ← CA1(比较/输出) ←┘
                                    ↑
                              EC(L3) → CA1 (直接通路)

记忆巩固(睡眠期): CA3/CA1 → 通过Sharp Wave Ripples → 新皮层(长期存储)
```

**核心参数：**
- 齿状回稀疏度: ~2% 激活率（极高的模式分离能力）
- CA3 循环连接: 约 1-2% 的 CA3→CA3 连接概率（自联想网络，每个 CA3 细胞接收约 1.2×10⁴ 个循环侧枝突触，Rolls 2013; Le Duigou et al. 2014）
- 记忆容量: 理论估算约 ~3.6×10⁴ 个独立模式 (Treves & Rolls 1991，基于 CA3 自联想网络理论，实际容量受稀疏度和编码方式影响)
- 巩固周期: 新记忆 → 海马依赖期数周到数月（某些情景记忆可达数年）→ 皮层独立 (Squire & Alvarez 1995)

#### 2.2.5 小脑 (Cerebellum) —— 预测与误差校正

| 编号 | 区域名称 | 生物功能 | 计算映射 | 神经元数量(估) | 特殊机制 |
|------|---------|---------|---------|--------------|---------|
| CB-01 | 小脑皮层-颗粒层 | 高维展开 | 输入维度扩展 | ~500-600亿 (Herculano-Houzel 2009) | 极高稀疏度，占小脑神经元>99% |
| CB-02 | 小脑皮层-浦肯野层 | 学习/抑制输出 | 预测误差学习 | ~0.15-0.26亿 (15-26M) | 每个接收~15-20万平行纤维突触(大鼠数据推算) |
| CB-03 | 深部核团 (DCN) | 小脑输出 | 预测信号输出 | ~0.05亿 | 兴奋性输出 |
| CB-04 | 下橄榄核 (IO) | 误差教学信号 | 监督误差信号 | ~0.01亿 | 攀缘纤维 |

**小脑计算模型：**
- 输入(苔藓纤维) → 颗粒细胞(高维稀疏展开) → 浦肯野细胞(学习加权)
- 下橄榄核提供 **教学信号**（攀缘纤维），驱动浦肯野细胞突触可塑性
- 本质上是一个 **在线监督学习器**，学习精确的前向预测模型

#### 2.2.6 神经调质系统 (Neuromodulatory Systems) —— 全局状态调控

| 编号 | 核团/系统 | 调质物质 | 生物功能 | 计算映射 | 影响范围 |
|------|----------|---------|---------|---------|---------|
| NM-01 | 腹侧被盖区 (VTA) | 多巴胺 (DA) | 奖励预测误差 | TD-error信号 | PFC, NAcc, 纹状体 |
| NM-02 | 黑质致密部 (SNc) | 多巴胺 (DA) | 运动/习惯强化 | 运动学习信号 | 背侧纹状体 |
| NM-03 | 蓝斑核 (LC) | 去甲肾上腺素 (NE) | 警觉/注意力/探索 | 探索-利用平衡 | 全脑广泛投射 |
| NM-04 | 中缝核 (Raphe) | 5-羟色胺 (5-HT) | 情绪/耐心/时间折扣 | 长期奖励折扣因子 | 全脑广泛投射 |
| NM-05 | 基底前脑 (BF) | 乙酰胆碱 (ACh) | 注意力/记忆编码 | 信噪比调节+可塑性开关 | 皮层, 海马体 |

**神经调质关键参数映射：**

```
多巴胺 (DA):
  - 相位性释放 → reward_prediction_error (RPE)
  - 紧张性水平 → motivation_level / exploration_bonus
  - D1受体 → 增强 Go 通路 (促进)
  - D2受体(Gi/o耦合) → 降低 D2-MSN 兴奋性 → 减弱 NoGo 信号 → 净效果为促进动作
  - DA 还通过突触前 D2 受体降低皮层→纹状体谷氨酸释放 (Surmeier et al. 2007)

去甲肾上腺素 (NE):
  - 低水平 → 困倦/不专注
  - 中等水平 → 聚焦注意力 (exploitation)
  - 高水平 → 高警觉/广泛注意力 (exploration)
  - 映射 → temperature 参数 (探索-利用权衡)

5-羟色胺 (5-HT):
  - 高水平 → 耐心/等待远期奖励
  - 低水平 → 冲动/偏好即时奖励
  - 映射 → discount_factor γ

乙酰胆碱 (ACh):
  - 高水平 → 增强自下而上处理 (learning_mode)
  - 低水平 → 增强自上而下处理 (inference_mode)
  - 映射 → learning_rate + signal_noise_ratio
```

## 三、全脑连接矩阵 (Connectome Overview)

### 3.1 主要通路汇总

```
信号流向图 (→ 为前馈, ⟷ 为双向, ⟶ 为调制)

[感觉输入]
    │
    ▼
[丘脑] ⟷ [初级感觉皮层] → [次级感觉皮层] → [联合皮层]
    ▲           │                                  │
    │           └──────────────────┐                │
    │                              ▼                ▼
[TRN]◄──────────────────── [后顶叶/颞叶] ⟷ [前额叶(PFC)]
                                  │                │
                                  ▼                ▼
                            [海马体] ⟷ [内嗅皮层]   [基底神经节]
                                  │                │
                                  ▼                ▼
                           [记忆巩固→皮层]     [丘脑→皮层(去抑制)]
                                                   │
                                                   ▼
                                             [运动皮层] → [运动输出]
                                                   ▲
                                                   │
                                              [小脑(校正)]

调制系统(虚线):
  VTA ⟶ PFC, NAcc, 纹状体 (多巴胺)
  LC  ⟶ 全脑 (去甲肾上腺素)
  Raphe ⟶ 全脑 (5-HT)
```

## 四、悟韵 (WuYun) 模块映射规划

### 4.1 脑区 → 计算模块映射

> **⚠️ 反作弊约束 (参见 00_design_principles.md)**
> - 所有新皮层脑区（S-\*、A-\*、M-\*）使用 **同一个 `CorticalColumn` 类**，仅通过 YAML 配置区分参数
> - 功能差异从参数差异 + 连接差异 + 学习经验中 **涌现**，不写入代码逻辑
> - 下表"核心算法"列描述的是 **物理机制**（硬件结构），不是硬编码的功能逻辑
> - 表中的"输入/输出"描述的是 **解剖连接拓扑**（先天布线），不是 IF-ELSE 路由

| 模块名称 | 对应脑区 | 硬件结构(可硬编码) | 功能差异来源(参数/学习) | 优先级 |
|---------|---------|-------------------|----------------------|-------|
| `CorticalColumn` | S-01~S-09 | 6层预测编码柱 | 配置: L4厚(感觉输入大), 初始连接→丘脑感觉核团 | P0 |
| `CorticalColumn` | A-01~A-11 | 6层预测编码柱 | 配置: L2/3厚+强循环连接+高DA受体密度 | P0 |
| `CorticalColumn` | M-01~M-04 | 6层预测编码柱 | 配置: L5厚(运动输出), 初始连接→脊髓/脑干 | P1 |
| `ThalamicRouter` | T-01~T-08 | 中继核团+TRN竞争抑制 | 路由偏好由皮层反馈权重学习决定 | P0 |
| `BasalGangliaEngine` | BG-01~BG-07 | D1/D2 MSN+Go/NoGo/Stop通路 | 动作选择由DA调制+权重学习决定 | P0 |
| `HippocampalMemory` | H-01~H-05 | DG稀疏化+CA3自联想+CA1比较 | 记忆内容由运行时编码决定 | P0 |
| `AmygdalaValence` | AM-01~AM-03 | BLA价值学习+CeA输出 | 什么是"威胁/奖励"由学习决定 | P1 |
| `CerebellumPredictor` | CB-01~CB-04 | 颗粒展开+浦肯野+攀缘纤维教学 | 预测模型由误差驱动学习形成 | P1 |
| `NeuromodulatorSystem` | NM-01~NM-05 | DA/NE/5-HT/ACh投射拓扑 | 调质释放量由神经元活动动态决定 | P0 |

> **注意**: 不再有 `PrefrontalExecutive` 作为独立模块。前额叶（A-01/A-03/A-04）只是参数不同的
> `CorticalColumn`，"执行控制/推理/规划"能力从强循环连接 + DA调制 + 与BG/海马的连接中涌现。

### 4.2 皮层柱内部 6 层结构规格

```
每个皮层柱 (CorticalColumn) 的统一内部结构:

┌─────────────────────────────────────────────┐
│ L1 - 分子层 (Molecular Layer)                │
│   功能: 接收远程反馈调制                       │
│   输入: 高层皮层的 top-down 预测               │
│   计算: 调制性加权 (不生成脉冲)                │
│   神经元类型: 极少(主要是轴突和树突)            │
├─────────────────────────────────────────────┤
│ L2/3 - 浅层锥体层 (Superficial Pyramidal)     │
│   功能: 特征关联 + 跨柱通信                    │
│   输入: L4 前馈 + L1 调制 + 侧向连接           │
│   计算: 预测误差计算 + 侧向竞争抑制             │
│   输出: → 同层其他柱(侧向), → 高层柱(前馈)     │
│   神经元: 兴奋性锥体 + 抑制性中间神经元          │
├─────────────────────────────────────────────┤
│ L4 - 颗粒层 (Granular Layer)                  │
│   功能: 主要输入层                             │
│   输入: 丘脑前馈信号 / 低层皮层前馈              │
│   计算: 局部特征提取                           │
│   输出: → L2/3 (向上), → L6 (向下)            │
│   神经元: 星形细胞(兴奋性) + 抑制性中间神经元    │
├─────────────────────────────────────────────┤
│ L5 - 深层锥体层 (Deep Pyramidal)              │
│   功能: 主要输出层 (驱动行为/皮层下)             │
│   输入: L2/3 + L6 + 直接丘脑输入               │
│   计算: 输出整合 + burst模式(注意力标记)        │
│   输出: → 基底神经节, → 脑干, → 脊髓            │
│         → 高层皮层(驱动性前馈)                  │
│   神经元: 大型锥体细胞 + 抑制性中间神经元        │
├─────────────────────────────────────────────┤
│ L6 - 多形层 (Polymorphic Layer)               │
│   功能: 反馈输出层 (→ 丘脑调制)                │
│   输入: L4 + L5 + 高层反馈                     │
│   计算: 生成 top-down 预测信号                  │
│   输出: → 丘脑(反馈调制), → 低层皮层L1          │
│   神经元: 多形锥体细胞 + 抑制性                 │
└─────────────────────────────────────────────┘
```
